This project aims at providing extensions to the StarUML modeling tool.
This very directory corresponds to the "modules" directory of StarUML
and in this first versions its contains the modules delivered with the
StarUML 5.0 version (the last one, the main product being not developed). 
Additional modules from the statuml catalog has been added as well.

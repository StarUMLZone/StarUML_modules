This module is an implementation of the UML 2.0 Profile named SPEM - Software Process Engineering Metamodel, developded by OMG(www.omg.org). 
The SPEM Profile supports the production of models for the modeling of software process, including roles, tasks, work products. 
The profile implements all features of the Final Adopted Specification (ptc/07-03-03) of OMG SPEM.
